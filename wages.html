Python Coding for Daily Wages Calculation


# wage calculation
days=float(input("Enter the number of days present:"))
wages=float(input("Enter wages per Day:"))
basic=wages*days;
HRA=basic*(10/100);
DA=basic*0.05;
PF=basic*0.12;
netsalary=basic+HRA+DA-PF;
print("\nBasic:%lf \nHRA:%f \nDA:%f \nPF:%f \nNet Salary:%f" %(basic,HRA,DA,PF,netsalary));
