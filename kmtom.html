Python Coding for Conversion of Km to M and Vice-Versa


#kilometre to metre
km=float(input("Enter the value in kilometre:"))
m=km*1000
print("{:.3f}".format(m),"is the value in metre.")

# metre to kilometre
m=float(input("Enter the value in metre:"))
km=m/1000
print("{:.3f}".format(km),"is the value in kilometre.")
